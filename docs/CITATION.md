# Citation

If you use this repository structure or derived workflow in a publication, please cite the associated archive paper.

The current archive paper corresponds to the initial OPOTEK-laser + Verasonics-focused implementation.
This repository scope is broader and intended to support additional hardware combinations over time.

## Recommended Citation Policy

1. Cite the archive paper for the initial OPOTEK + Verasonics method and workflow context.
2. Optionally cite this repository URL in methods or software sections.
