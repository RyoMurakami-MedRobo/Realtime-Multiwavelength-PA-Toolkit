# Sample Data

This folder contains a tiny educational mock dataset for dry-run checks.

## Files

- `create_mock_dataset.m`: generator script
- `mock_dataset.mat`: generated tiny package data
- Shared-memory demo file path is `shared_image_data_public.dat` under the system temp folder.

## Regenerate

In MATLAB:

```matlab
cd sample_data
create_mock_dataset
```
